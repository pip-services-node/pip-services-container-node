Static readFromFile

  • readFromFile(correlationId: string, path: string, parameters: ConfigParams): ContainerConfig
    • Defined in config/ContainerConfigReader.ts:34

    Static method that reads the JSON/YAML data from the file and returns it as a parameterized ContainerConfig object. If the target file's extention is not ".json", ".yaml", or ".yml", then the reader will default to using the readFromJsonFile method.

    throws

    a ConfigException if the path is null.
